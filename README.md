## SH-Cloner

## Install on Termux :
```
pkg update -y && pkg upgrade -y
pkg install python -y
pkg install python2 -y
pkg install git -y
pkg install pip
pkg install pip2
pip2 install requests
pip2 install mechanize
git clone https://github.com/ShHasib/OLD-CLONER
```
## Run On Termux :
```
cd OLD-CLONER
python2 cloner.py
```
